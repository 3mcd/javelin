    Introduction

    Javelin is a suite of packages used to build multiplayer games. The core package is an Entity-Component System (ECS).

    See a live, multiplayer game example. Connect with a friend and move around using the WASD keys!

    Check out the code at the GitHub repo.

    Or, move onto the ECS chapter to start learning more.
